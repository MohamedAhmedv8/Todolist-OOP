# 📝 PHP OOP Task Manager

A simple **Task Management System** built with **PHP** using **Object-Oriented Programming (OOP)** principles.  
This project demonstrates clean code organization and the implementation of core OOP concepts such as classes, objects, encapsulation, and reusable methods.

- ✅ Add new tasks
- ✏️ Edit existing tasks
- 🗑️ Delete tasks
- ✔️ Mark tasks as completed
- 📋 Display all tasks

## 🚀 Getting Started

### Clone the repository

```bash
git clone https://github.com/MohamedAhmedv8/Todolist-OOP.git
```

### Open the project

Move the project into your local server directory (XAMPP, Laragon, etc.) and start your server.

Then open:

```
http://localhost/Todolist-OOP
```
